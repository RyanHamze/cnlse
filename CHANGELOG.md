# Changelog

## 0.1.0 - 2026-06-30

- Created initial open-source Python package.
- Added attribution, citation, license, and project plan.
- Ported small MATLAB utilities: `nmod`, `fastshift`, `fastexp`, `phi2pow`.
- Added OOK signal generation, laser source, and Mach-Zehnder modulator.
- Added explicit simulation state, field creation, and average power helpers.
- Added first scalar fiber propagation support with attenuation, GVD, and SPM.
- Added plotting helper, example script, tutorials, and Streamlit app.
- Included legacy MATLAB source for reference and attribution.
