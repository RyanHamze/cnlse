# Legacy MATLAB Source

This folder preserves the original MATLAB simulation files from the 2009 senior project:

**Design and Implementation of Fiber Optics Using MATLAB**  
Arts, Sciences, and Technology University in Lebanon  
Beirut, Lebanon, October 2009

Original authors:

- Ryan Maher Hamze
- Krayem Ahmad Ajjawi
- Meriam Hawwa

The Python package in `src/cnlse` is a modern open-source implementation and continuation of this work. The MATLAB files are included for historical reference, comparison, and attribution.

Related ebook:

**Design, Implementation and Simulation of Fiber Optics Modulation Using MATLAB**  
ASIN: `B077Z6FMF3`  
2nd edition, published December 6, 2017
