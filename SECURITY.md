# Security Policy

`cnlse` is an educational simulation package and should not be used as a safety-critical engineering control system.

## Reporting Issues

Please report security or supply-chain concerns through the project maintainer:

- Website/contact: [ryanhamze.com](https://ryanhamze.com)

## Scope

Security-sensitive areas include:

- package installation metadata;
- GitHub workflow configuration;
- generated files or examples that read/write local paths;
- future GUI or web app additions.
